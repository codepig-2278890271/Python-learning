# 🐍 Python 学习项目

从零开始学 Python，边学边做一个可玩的横版跳跃游戏！

---

## 📁 项目结构

```
Python-learning/
├── basic/                     # 📖 Python 零基础入门（12课）
│   ├── 01_认识Python与第一个程序.py
│   ├── 02_变量与数据类型.py
│   ├── 03_运算符与表达式.py
│   ├── 04_用户输入与条件判断.py
│   ├── 05_循环与控制流.py
│   ├── 06_字符串操作.py
│   ├── 07_列表元组字典集合.py
│   ├── 08_函数.py
│   ├── 09_文件操作.py
│   ├── 10_面向对象编程入门.py
│   ├── 11_异常处理.py
│   └── 12_综合练习与小项目.py
│
└── project/                   # 🎮 超级小猫 —— Pygame 横版跳跃游戏
    ├── src/                   # 源代码
    │   ├── main.py            # 游戏引擎 & 主循环
    │   ├── settings.py        # 配置常量（调数值来这里！）
    │   ├── sprites.py         # 精灵类（玩家、敌人、金币等）
    │   ├── level.py           # 关卡数据定义
    │   └── ui.py              # 界面：菜单、HUD、暂停面板
    ├── docs/                  # 文档
    ├── scripts/               # 编译打包脚本
    ├── build/                 # 编译产物
    └── log/                   # 运行日志
```

---

## 🚀 快速开始

### 学习 Python 基础

从 `basic/` 目录开始，按编号顺序逐个运行：

```bash
# 运行第一个学习文件
python3 basic/01_认识Python与第一个程序.py
```

> 💡 建议用 VS Code 打开文件，逐段运行，边读边敲！

### 运行游戏

```bash
# 安装依赖
pip3 install pygame

# 启动游戏
cd project
python3 -m src.main
```

## 🎮 超级小猫 —— 游戏介绍

一个类超级玛丽风格的横版跳跃游戏，使用 **Pygame** 开发，适合作为 Python 学习项目长期迭代改进。

### 操作说明

| 按键 | 功能 |
|------|------|
| A / D | 左右移动 |
| K | 跳跃 |
| J | 发射子弹 |
| ESC | 暂停 / 继续 |

### 游戏目标

- 👾 踩敌人头顶或用子弹消灭敌人
- 🪙 收集金币获得分数
- 🏁 到达关卡最右侧的旗帜即可通关
- ❤️ 初始 3 条命，碰到敌人侧面或掉入坑中会扣命

### 编译为可执行文件

```bash
bash project/scripts/build.sh
```

## 📚 学习路线

1. **先学基础**：按 `basic/` 目录下编号顺序学完 12 个文件，掌握 Python 核心语法
2. **再读项目**：打开 `project/` 目录，从 `main.py` 开始阅读游戏代码
3. **动手修改**：参考 `project/docs/代码分析指南.md`，尝试调整游戏参数或添加新功能
4. **持续迭代**：给游戏加新关卡、新敌人、音效……想加什么加什么！
